# flask-auth-ex1

## Install

    git clone https://github.com/evadelzz1/flask-auth-ex1.git

    cd ./flask-auth-ex1/

    python -m venv .venv && source .venv/bin/activate

    pip install -r requirements.txt

    python app.py
